# デモ用FAQ

この資料は、通信代理店向け「店舗スタッフ確認AIデモ」のための架空FAQです。
実際の料金、契約条件、キャンペーン、法令判断を保証するものではありません。

## 1. 本人確認書類

個人契約の場合、本人確認書類として以下のいずれかを確認する想定です。

- 運転免許証
- マイナンバーカード
- 在留カード
- パスポート

ただし、最新の必要書類や本人確認方法は、実際の運用資料に従う必要があります。

## 2. 未成年契約

18歳未満の方が申し込みを行う場合、親権者の同意が必要です。

親権者同意の取得方法、必要書式、同席要否については、契約内容や社内運用により異なる可能性があります。

資料に詳細がない場合は、本部確認が必要です。

## 3. 法人契約

法人契約では、以下の書類が必要になる場合があります。

- 法人確認書類
- 担当者の本人確認書類
- 委任状
- 登記情報または会社情報を確認できる資料

ただし、必要書類は契約種別や申込内容によって異なるため、資料に明記がない場合は本部確認が必要です。

## 4. 解約手続き

解約手続きは、原則として解約希望月の月末までに申し出る必要があります。

最低利用期間が設定されている場合、その期間を満了しない解約には費用が発生する可能性があります。

具体的な解約費用や締切日は契約内容により異なるため、本部確認が必要です。

## 5. 初期契約解除

一部の契約では、契約書面の受領日から一定期間内であれば、初期契約解除が可能な場合があります。

ただし、対象契約、申請方法、期限、費用負担の有無は契約内容により異なります。

資料に詳細がない場合は、本部確認が必要です。

## 6. オプション加入

オプション加入が必須か任意かは、契約プランやキャンペーン条件により異なります。

資料に「必須」と明記されていない場合、店舗スタッフの判断で必須と案内してはいけません。

不明な場合は本部確認が必要です。

## 7. キャンペーン併用

キャンペーンの併用可否は、対象条件により異なります。

キャンペーン資料に併用可否が明記されている場合のみ、その範囲で案内できます。

資料に記載がない場合は、本部確認が必要です。
