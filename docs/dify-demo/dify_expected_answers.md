# Dify期待回答例

Difyチャットボットが返すべき回答の例を示します。
テスト時に実際の回答と比較して確認してください。

## Q1. 未成年の方は契約できますか？

### 期待回答（模範例）

```
【結論】
デモ資料上で確認できる範囲では、18歳未満の方の契約には親権者の同意が必要です。

【説明】
未成年契約では、親権者の同意が必要とされています。
また、親権者同意の形式、必要書類、同席の必要性については、社内運用に従って確認する必要があります。

【注意点】
親権者同意の具体的な取得方法や必要書式については、資料上で確認できないため、本部確認が必要です。

【本部確認】
必要

## Q5. 解約金は発生しますか？

### 期待回答（模範例）

```
【結論】
デモ資料上では、最低利用期間内の解約に費用が発生する可能性があります。

【説明】
最低利用期間が設定されている場合、その期間を満了しない解約には費用が発生する可能性があります。
ただし、具体的な解約費用や最低利用期間の詳細は、契約内容により異なります。

【注意点】
個別の契約内容によって、解約金の有無や金額が変わる可能性があります。
店舗スタッフの判断で「費用が発生しません」と案内してはいけません。

【本部確認】
必要（個別の契約条件を確認する場合）

## Q7. キャンペーンは併用できますか？

### 期待回答（模範例）

```
【結論】
デモ資料上では、キャンペーンの併用可否は各キャンペーン条件により異なるとされています。

【説明】
併用可能と明記されている場合のみ、併用可能として案内できます。
資料に併用可否の記載がない場合は、本部確認が必要です。

【注意点】
店舗スタッフの判断で、併用可能と断定して案内してはいけません。
「この2つのキャンペーンは併用できます」と答える場合は、必ず資料に併用可と明記されていることを確認してください。

【本部確認】
資料に併用可否の記載がない場合は必要

【参照資料】
デモ用キャンペーン条件表
```

## Q13. お客様にどこまで案内してよいですか？

### 期待回答（模範例）

```
【結論】
デモ資料に記載されている範囲のみ案内でき、資料にない内容については本部確認を案内する必要があります。

【説明】
店舗スタッフは、以下の範囲で案内できます。

- 資料に明記されている料金、契約期間、キャンペーン条件
- 資料に記載されている必要書類や手続きの流れ
- 資料に明記されている「本部確認が必要」という指示

資料に明記されていない内容については、独自判断で案内してはいけません。

【注意点】
お客様から受けた質問に対して、資料に記載がない場合、店舗スタッフは推測で答えるのではなく、本部確認が必要であることを案内してください。

【本部確認】
資料にない内容を案内する場合は必要

【参照資料】
デモ用FAQ、デモ用重要事項説明書

## 期待回答の共通特性

良い回答は以下の特性を持ちます。

### ✅ 好ましい特性

- [ ] 資料に記載された情報のみを使用している
- [ ] 不確実な内容は「本部確認が必要」と案内している
- [ ] 店舗スタッフが困らない具体的な説明がある
- [ ] 法令判断や料金保証をしていない
- [ ] 営業的な強制がない
- [ ] 個人情報への警告がある（必要な場合）

### ❌ 避けるべき特性

- [ ] 資料に記載されていない情報を推測で答えている
- [ ] 「できます」「対象です」などと断定している
- [ ] 「必ず」「絶対」などの強い表現がある
- [ ] 法令判断をしている
- [ ] 料金や契約可否を保証している
- [ ] 営業トーク的表現がある

---

## テスト時の比較方法

Difyの実際の回答を得た後：

1. **結論部分**を確認：資料から正しい情報が引き出せているか
2. **説明部分**を確認：根拠が明確か、推測が含まれていないか
3. **本部確認項目**を確認：「本部確認が必要」という指示があるか
4. **参照資料**を確認：適切な資料を参照しているか
5. **禁止事項**を確認：断定的表現がないか、営業トークがないか

いずれかで違和感がある場合は、システムプロンプトの修正またはDifyの学習データ設定を見直してください。
