# デモ用キャンペーン条件表

この資料は、通信代理店向け「店舗スタッフ確認AIデモ」のための架空キャンペーン条件表です。
実際のキャンペーン条件を保証するものではありません。

## 1. 新規契約キャンペーン

**対象：**
新規契約のお客様

**主な条件：**
- 指定プランへの加入
- 申込月を含む一定期間の継続利用
- 本人確認書類の確認完了

**注意点：**
- 他キャンペーンとの併用可否は、キャンペーンごとに異なります。
- 資料に併用可否の記載がない場合は、本部確認が必要です。

## 2. 乗り換えキャンペーン

**対象：**
他社サービスから乗り換えのお客様

**主な条件：**
- 乗り換え元サービスの確認
- 指定期間内の申込
- 本人確認書類の確認完了

**注意点：**
- 乗り換え元サービスの条件によって対象外になる場合があります。
- 資料上で対象可否を判断できない場合は、本部確認が必要です。

## 3. 家族割引キャンペーン

**対象：**
同一住所または家族関係が確認できるお客様

**主な条件：**
- 家族関係を確認できる資料
- 同一グループでの申込
- 指定プランへの加入

**注意点：**
- 家族関係の確認方法は契約内容により異なる場合があります。
- 資料に詳細がない場合は、本部確認が必要です。

## 4. オプション加入特典

**対象：**
指定オプションに加入したお客様

**主な条件：**
- 指定オプションへの加入
- 一定期間の継続利用
- キャンペーン期間内の申込

**注意点：**
- オプション加入が必須か任意かは、キャンペーン条件により異なります。
- 資料に「必須」と明記されていない場合、必須と断定して案内してはいけません。
- 不明な場合は本部確認が必要です。

## 5. キャンペーン併用

キャンペーンの併用可否は、各キャンペーン条件により異なります。

併用可能と明記されている場合のみ、併用可能として案内できます。

資料に記載がない場合は、本部確認が必要です。
